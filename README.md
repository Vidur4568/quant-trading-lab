# Quant Trading Lab

## Overview
This repository documents my journey of learning quantitative research and building a quantitative trading system from scratch using Python. It contains course notebooks, experiments, and projects focused on financial data analysis, feature engineering, machine learning, and systematic trading strategies.

## Goals
- Learn quantitative research methods
- Build systematic trading strategies
- Develop machine learning models for financial markets
- Create a backtesting engine
- Analyze strategy performance and risk
- Build a professional quantitative finance portfolio

## Project Structure

- **src/** – Core Python modules for data processing, feature engineering, trading strategies, and backtesting.
- **data/** – Raw and processed market data used for research and model development.
- **notebooks/** – Jupyter notebooks containing course exercises, experiments, and quantitative research.
- **results/** – Backtest results, performance metrics, charts, and strategy outputs.
- **docs/** – Project documentation, methodology, notes, and research summaries.
