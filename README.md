# quant-trading-lab
## Overview
This repository documents my journey building a quantitative trading system from scratch using Python.

## Goals
- Learn quantitative research methods
- Build systematic trading strategies
- Develop machine learning models for financial markets
- Create a backtesting engine
- Analyze performance and risk

## Project Structure
- src/ - Core Python modules
- data/ - Raw and processed market data
- notebooks/ - Research and experiments
- results/ - Backtest outputs and charts
- docs/ - Notes and methodology
